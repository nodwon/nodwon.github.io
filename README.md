## (쉘 스크립트 관련) getopt 및 getopts 명령어
### getopt란?
**getopt 함수는 C인터페이스를 기반으로 POSIX 유틸리티 구문 지침을 따르는 명령**
### getopts란?
**명령 행 인수를 구문 분석하기 위한 내장 Unix 쉘 명령**
**command-line option을 분석하는데 사용**
